Ngày 14/7 , Trung_tâm Giao_dịch chứng_khoán Hà_Nội chính_thức hoạt_động với phiên giao_dịch đầu_tiên có trị_giá hơn 8 tỉ đồng với 622.854 cổ_phiếu ( CP ) được giao_dịch . Giá giao_dịch của các CP trong phiên giao_dịch đầu_tiên này đều đạt khá cao so với dự_kiến : GHA ( Công_ty cổ_phần Giấy Hải_Âu ) đạt 16.000 đồng / CP ; HSC ( Công_ty cổ_phần Hacinco ) : 40.000 đồng / CP ; KHP ( Công_ty cổ_phần Điện_lực Khánh_Hòa ) : 25.000 đồng / CP ; VTL ( Công_ty cổ_phần Thăng_Long ) : 22.500 đồng / CP ; VSH ( Công_ty cổ_phần Thủy_điện Vĩnh_Sơn Sông_Hinh ) : 20.000 đồng / CP. Giá CP thấp nhất là CID ( Công_ty cổ_phần Xây_dựng và Phát_triển cơ_sở_hạ_tầng ) đạt 15.000 đồng / CP .
