World_Cup 2006 : Danh_sách 23 trọng_tài chính_thức Trọng_tài Markus_Merk của nước chủ nhà Đức_LĐBĐ thế_giới ( FIFA ) đã gây bất_ngờ cho giới chuyên_môn khi chỉ dùng 23 trọng_tài phục_vụ cho tất_cả 64 trận đấu tại World_Cup 2006 ở Đức vào mùa hè này . Ủy ban_trọng_tài đã chọn_lựa trong số 44 ứng_cử_viên sau khi đã có các cuộc kiểm_tra tại Frankfurt cuối tuần trước . 23 trọng_tài đến từ 23 quốc_gia khác nhau . Còn nhớ tại World_Cup 2002 ở Nhật_Bản và Hàn_Quốc thì đã có 36 trọng_tài làm nhiệm_vụ trong 64 trận . Và FIFA đã trông_chờ sẽ chọn được khoảng 30 trọng_tài lần này . Tuy_nhiên , chỉ có 23 trọng_tài đủ tiêu_chuẩn mà FIFA đã lựa_chọn . Có hai trọng_tài được bổ_sung là Frank De_Bleeckere ( Bỉ ) và Carlos_Batres ( Guatemala ) , bởi họ đã hoàn_thành các các bài kiểm_tra mà không vượt qua được cuộc kiểm_tra tại Frankfurt trước đó vì chấn_thương . World_Cup sẽ bắt_đầu vào ngày 9/6 tại Munich khi chủ nhà Đức_tiếp Costa_Rica . Trận chung_kết tại Berlin vào ngày 9/7 . Danh_sách 23 trọng_tài được chọn làm nhiệm_vụ tại World_Cup : Essam_Abd El_Fatah ( Ai_Cập ) Carlos_Amarilla ( Paraguay ) Benito_Archundia ( Mexico ) Carlos_Batres ( Guatemala ) Massimo_Busacca ( Thụy_Sĩ ) Coffi_Codjia ( Benanh ) Frank De_Bleeckere ( Bỉ ) Massimo_de Santis ( Italia ) Horacio_Elizondo ( Argentina ) Valentin_Ivanov ( Nga ) Toru_Kamikawa ( Nhật_Bản ) Jorge_Larrionda ( Uruguay ) Shamsul_Maidin ( Singapore ) Manuel Mejuto_Gonzalez ( Tây_Ban_Nha ) Markus_Merk ( Đức ) Lubos_Michel ( Slovakia ) Graham_Poll ( Anh ) Eric_Poulat ( Pháp ) Peter_Prendergast ( Jamaica ) Oscar_Ruiz ( Colombia ) Mark_Shield ( Australia ) Carlos_Simon ( Brazil ) Kyros_Vassaras ( Hy_Lạp )
