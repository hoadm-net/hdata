15 Việt_kiều được bình chọn danh_hiệu Vinh_Danh_Nước Việt 2005 Ngày 28-12 , báo_điện_tử VietNamNet chính_thức công_bố 15 gương_mặt Việt_kiều được chọn trao danh_hiệu Vinh_danh_Nước Việt 2005 . Những gương_mặt Việt_Kiều được bình chọn Vinh_Danh đất Việt 2005 Đó là TS. Nguyễn_Quốc_Bình ( VK Canada ) ; TS. Nguyễn_Trọng_Bình ( VK_Mỹ ) ; PGS , TS. Nguyễn_Lương_Dũng ( VK_Đức ) ; TS. Nguyễn_Trí_Dũng ( VK_Nhật ) ; Họa_sĩ Lê_Bá_Đảng ( VK Pháp ) ; GS , TS. Nguyễn_Quý_Đạo ( VK Pháp ) ; Nhạc_sỹ Nguyễn_Thiện_Đạo ( VK Pháp ) , Bác_sĩ Bùi_Minh_Đức ( VK_Mỹ ) ; TS. Lê_Phước_Hùng ( VK_Mỹ ) ; ThS. Phạm_Đức_Trung_Kiên ( VK_Mỹ ) ; GS , TS. Đoàn_Kim_Sơn ( VK Pháp ) ; GS Toán_học Lê_Tự_Quốc_Thắng ( VK_Mỹ ) ; Ông Phan_Thành - Chủ_tịch Hiệp_hội Doanh_nghiệp Việt_Kiều ( VK Canada ) ; GS , TS Nguyễn_Văn_Tuấn ( VK_Úc ) và TS nhạc_sĩ Nguyễn_Lân_Tuất ( VK_Nga ) . Buổi lễ trao danh_hiệu " Vinh_Danh_Nước Việt – 2005 " do báo_điện_tử VietNamNet và Đài_Truyền_hình Việt_Nam phối_hợp tổ_chức sẽ diễn ra tại Văn_Miếu - Quốc_Tử_Giám , Hà_Nội vào hồi 19 h15 ngày Chủ_nhật 05/02/2006 ( tức ngày 8 Tết Bính_Tuất ) . Chương_trình sẽ được truyền_hình trực_tiếp trên sóng VTV1_Đài_Truyền_hình Việt_Nam .
