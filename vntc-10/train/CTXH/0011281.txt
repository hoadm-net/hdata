Đề_xuất thêm 25 tuyến đường bắt_buộc đội mũ_bảo_hiểm ( NLĐ ) - Nhằm hạn_chế sự_cố cho người đi xe hai bánh , Khu Quản_lý Giao_thông đô_thị số 1 sẽ trình Sở GTCC và Ban An_toàn Giao_thông TP một_số tuyến đường bắt_buộc đội mũ_bảo_hiểm . Theo dự_kiến , sẽ có 25 tuyến đường : Nguyễn_Tất_Thành ( từ cầu Khánh_Hội đến cầu Tân_Thuận ) - quận 4 ; Tôn_Đức_Thắng ( từ Nguyễn_Tất_Thành đến Lê_Thánh_Tôn ) - quận 1 ; Lê_Thánh_Tôn ( từ Tôn_Đức_Thắng - Nguyễn_Hữu_Cảnh ) - quận 1 ; Nguyễn_Hữu_Cảnh ( từ Lê_Thánh_Tôn - Điện_Biên_Phủ ) - quận 1 , Bình_Thạnh ; Điện_Biên_Phủ ( từ cầu Sài_Gòn đến Đinh_Tiên_Hoàng ) - quận 1 , Bình_Thạnh ; Đinh_Tiên_Hoàng ( từ Điện_Biên_Phủ đến Võ_Thị_Sáu ) - quận 1 ; Võ_Thị_Sáu ( từ Nguyễn_Phi_Khanh đến Cách_Mạng_Tháng_Tám ) - quận 1 , 3 ; đường 3 Tháng 2 ( từ Cách_Mạng_Tháng_Tám đến vòng xoay Cây_Gõ ) - quận 10 ; Bạch_Đằng ( từ Lê_Quang_Định đến Xô_Viết_Nghệ_Tĩnh ) - quận Bình_Thạnh ; Phan_Đăng_Lưu ( từ Lê_Quang_Định đến ngã tư Phú_Nhuận ) - quận Bình_Thạnh , Phú_Nhuận ; Hoàng_Văn_Thụ ( từ ngã tư Phú_Nhuận đến Lăng_Cha_Cả ) - quận Phú_Nhuận ; Trường_Chinh ( từ ngã tư Bảy_Hiền đến Quốc_lộ 1A ) - quận 12 , Tân_Bình ; Cộng_Hòa ( từ Trường_Chinh đến Hoàng_Văn_Thụ ) - quận Tân_Bình ; Tây_Thạnh ( từ Trường_Chinh đến Lê_Trọng_Tấn ) - quận Tân_Phú ; Lê_Văn_Việt ( từ ngã tư Thủ_Đức đến Hoàng_Hữu_Nam ) - quận 9 ; Quang_Trung ( từ vòng xoay ngã sáu đến cầu Chợ_Cầu ) - quận Gò_Vấp ; Nguyễn_Oanh ( từ ngã sáu Gò_Vấp đến cầu An_Lộc ) - quận Gò_Vấp ; Hà_Huy_Giáp ( từ cầu An_Lộc đến cầu Phú_Long ) - quận 12 ; Kinh_Dương_Vương ( từ vòng xoay Cây_Gõ đến ngã ba An_Lạc ) - quận 6 , Tân_Bình ; Tô_Ký ( Quốc_lộ 1A đến ngã ba Chùa ) - quận 12 , huyện Hóc_Môn ; Nguyễn_Ảnh_Thủ ( Lê_Văn_Khương đến Tô_Ký ) - quận 12 , huyện Hóc_Môn ; Hương_lộ 2 ( từ Tỉnh_lộ 8 đến Quốc_lộ 22 ) - huyện Củ_Chi ; Tỉnh_lộ 15 ( từ cầu Xáng đến Bến_Súc ) - huyện Củ_Chi ; Tỉnh_lộ 7 ( từ cầu Tân_Thái đến Tỉnh_lộ 15 ) - huyện Củ_Chi ; Nguyễn_Hữu_Thọ ( từ Nguyễn_Văn_Linh đến Nguyễn_Văn_Tạo ) - quận 7 , huyện Nhà_Bè .
